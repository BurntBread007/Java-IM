PowerShell:

1. Shift + Right-Click on open space in your folder, and select "Open PowerShell window here".
2. Copy this: "java -jar Client.jar" (without the " of course).
3. Paste it into the PowerShell window with either Ctrl+V or Right-Click.

Windows Command Prompt:
Double-click on the corresponding .bat file.


Command List:
list : Lists the usernames connected.
leave : Disconnects yourself.
fart : Fart.
version : Displays version running on server.
ip : Displays IP of server.
port : Displays port # of server.
credit : Credits to the devs.
help : This.
rename : Changes your username to the new one given.
(WIP) kick : Disconnects another user.
coinflip : Flips a coin.
